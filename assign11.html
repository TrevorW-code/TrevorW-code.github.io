Final Remarks

The most important thing to note here is the inverse relationship between Horsepower and Acceleration. While an engine with more horsepower seems like it would be faster, using data, this is clearly not the case.

Second Basic Vega Illustration

Using the "stocks.csv" data, I wanted to plot a few different companies stock data

  • x-axis: "Date"
  • y-axis: "Price"
  • color: "Company"

Embeded Vega

Vega Code


            {
              "width": 300,
              "height": 200,
              "data": {
                "url": "https://vega.github.io/vega-datasets/data/stocks.csv"
              },
              "mark": {"type": "line"},
              "encoding": {
                "x": {"field": "date",
                      "type": "temporal"
                },
                "y": {"field": "price", "type": "quantitative"},
                "color": {"field": "symbol", "type": "nominal"}
                }
              };
